## [Resume (React, Redux, Webpack2)](https://maxislav.github.io/resume2/dist/#/itskill)

## Develop

### Install
define you skill <b>./src/asset/skill-history.json</b>


```
~$ mpm i -g webpack
~$ mpm i -g webpack-dev-server
~$ mpm i
```

***
### Debug
```
~$ npm run dev
```

### Build prod
```
~$ npm run prod
```
Download files from ~/dist/* to you work dir
